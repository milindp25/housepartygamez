# Plan 8: Mafia Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Classic Mafia: secret roles (Mafia, Doctor, Detective, Civilians), night actions, day discussion and elimination votes, town-vs-mafia win conditions. The most complex state machine in the catalog — built last, exactly as the spec ordered.

**Architecture:** Still one `GameDefinition` — the engine needs no changes (this was the design bet, and this plan proves it). All secrecy lives in `playerView`: mafia see each other, the detective sees their own investigation log, the host/TV sees no roles until players are eliminated. Role assignment is hash-seeded from `init`'s `now` (pure + unpredictable, as plans 6–7). Mafia has no prompts — `init` receives an empty prompt array.

**v1 rules (Classic mode per spec; Chaos/Jester/etc. are later modes):**
- Roles by player count `n`: mafia = `max(1, floor(n / 4))`, 1 Doctor, 1 Detective, rest Civilians.
- **Night:** each mafioso votes a victim (plurality; deterministic hash tie-break). Doctor protects anyone (self allowed). Detective investigates one player → learns mafia / not mafia. Night ends when all living role-holders acted, or the timer fires (missing actions are skipped).
- **Day:** the night's outcome is announced ("X was eliminated" / "the Doctor saved someone!"), discussion timer runs, then a public elimination vote (no self-votes; plurality; tie → nobody eliminated). Eliminated players' roles are revealed and they spectate.
- **Win:** town wins when all mafia are gone; mafia wins when mafia ≥ everyone else alive.

**Standards / Pre-flight:** as prior plans; requires `plan-3-four-games` (independent of 4–7, but recommended last — it's the hardest to verify manually and benefits from a battle-tested platform).

```ts
import type { GameDefinition, GamePlayer, TimedState } from '../engine/types'

export type MafiaRole = 'mafia' | 'doctor' | 'detective' | 'civilian'

export interface MafiaSettings {
  nightSeconds: number
  discussionSeconds: number
  voteSeconds: number
  revealSeconds: number
}

/** Mafia uses no prompt content; the prompt type is never instantiated. */
export type MafiaPrompt = never

export interface MafiaState extends TimedState {
  phase: 'night' | 'day' | 'vote' | 'reveal' | 'finished'
  /** 1-based day counter; increments each dawn. */
  day: number
  players: GamePlayer[]
  roles: Record<string, MafiaRole>
  alive: Record<string, boolean>
  /** Current-night actions, cleared at dawn. */
  night: {
    mafiaVotes: Record<string, string>
    doctorSave: string | null
    detectiveCheck: string | null
  }
  lastNight: { killedId: string | null; saved: boolean } | null
  /** Only ever surfaced in the detective's own playerView. */
  detectiveLog: Array<{ targetNickname: string; isMafia: boolean }>
  /** Current-day elimination votes. */
  votes: Record<string, string>
  lastVote: {
    eliminatedId: string | null
    revealedRole: MafiaRole | null
    tally: Array<{ nickname: string; count: number }>
  } | null
  winner: 'town' | 'mafia' | null
  seed: number
  settings: MafiaSettings
}

interface AlivePlayer {
  id: string
  nickname: string
  alive: boolean
}

interface BaseView {
  day: number
  players: AlivePlayer[]
}

export type MafiaPlayerView = BaseView & {
  role: MafiaRole
  isAlive: boolean
  /** mafia only: nicknames of the whole mafia team */
  mafiaTeam?: string[]
  /** detective only: their investigation history */
  detectiveLog?: Array<{ targetNickname: string; isMafia: boolean }>
} & (
    | {
        phase: 'night'
        /** null when this player has no night action (or is dead): "the town sleeps" */
        action: 'kill' | 'save' | 'investigate' | null
        candidates: Array<{ id: string; nickname: string }>
        yourTarget: string | null
        deadline: number | null
      }
    | { phase: 'day'; lastNight: { killedNickname: string | null; saved: boolean }; deadline: number | null }
    | {
        phase: 'vote'
        candidates: Array<{ id: string; nickname: string }>
        yourVote: string | null
        deadline: number | null
      }
    | {
        phase: 'reveal'
        eliminatedNickname: string | null
        revealedRole: MafiaRole | null
        tally: Array<{ nickname: string; count: number }>
      }
    | { phase: 'finished'; winner: 'town' | 'mafia'; allRoles: Array<{ nickname: string; role: MafiaRole }> }
  )

export type MafiaHostView = BaseView &
  (
    | { phase: 'night'; actionsDone: number; actionsNeeded: number; deadline: number | null }
    | { phase: 'day'; lastNight: { killedNickname: string | null; saved: boolean }; deadline: number | null }
    | { phase: 'vote'; votedCount: number; totalVoters: number; deadline: number | null }
    | {
        phase: 'reveal'
        eliminatedNickname: string | null
        revealedRole: MafiaRole | null
        tally: Array<{ nickname: string; count: number }>
      }
    | { phase: 'finished'; winner: 'town' | 'mafia'; allRoles: Array<{ nickname: string; role: MafiaRole }> }
  )

function hash(s: string): number {
  let h = 0
  for (let i = 0; i < s.length; i++) h = (h * 31 + s.charCodeAt(i)) | 0
  return h >>> 0
}

/**
 * Seeded role assignment: shuffle players by hash(id + seed), deal mafia first,
 * then doctor, detective, civilians. Pure and reproducible; unpredictable at
 * the table because seed = game start time.
 */
function assignRoles(players: GamePlayer[], seed: number): Record<string, MafiaRole> {
  const shuffled = [...players].sort((a, b) => hash(a.id + seed) - hash(b.id + seed))
  const mafiaCount = Math.max(1, Math.floor(players.length / 4))
  const roles: Record<string, MafiaRole> = {}
  shuffled.forEach((p, i) => {
    roles[p.id] =
      i < mafiaCount ? 'mafia' : i === mafiaCount ? 'doctor' : i === mafiaCount + 1 ? 'detective' : 'civilian'
  })
  return roles
}

function nickname(state: MafiaState, id: string | null): string | null {
  if (id === null) return null
  return state.players.find((p) => p.id === id)?.nickname ?? '?'
}
function aliveIds(state: MafiaState): string[] {
  return state.players.filter((p) => state.alive[p.id]).map((p) => p.id)
}
function aliveWithRole(state: MafiaState, role: MafiaRole): string[] {
  return aliveIds(state).filter((id) => state.roles[id] === role)
}

function checkWinner(state: MafiaState): 'town' | 'mafia' | null {
  const mafiaAlive = aliveWithRole(state, 'mafia').length
  const othersAlive = aliveIds(state).length - mafiaAlive
  if (mafiaAlive === 0) return 'town'
  if (mafiaAlive >= othersAlive) return 'mafia'
  return null
}

/** Plurality with deterministic hash tie-break so night resolution is pure. */
function plurality(votes: Record<string, string>, seed: number): string | null {
  const counts = new Map<string, number>()
  for (const t of Object.values(votes)) counts.set(t, (counts.get(t) ?? 0) + 1)
  let best: string | null = null
  for (const [id, n] of counts) {
    if (best === null || n > counts.get(best)! || (n === counts.get(best)! && hash(id + seed) < hash(best + seed))) {
      best = id
    }
  }
  return best
}

/** All living role-holders have acted (connected ones, so a dropped phone never stalls the night). */
function nightComplete(state: MafiaState): boolean {
  const need = (role: MafiaRole) =>
    aliveWithRole(state, role).filter((id) => state.players.find((p) => p.id === id)?.connected)
  const mafiaDone = need('mafia').every((id) => state.night.mafiaVotes[id])
  const doctorDone = need('doctor').length === 0 || state.night.doctorSave !== null
  const detectiveDone = need('detective').length === 0 || state.night.detectiveCheck !== null
  return mafiaDone && doctorDone && detectiveDone
}

/** Dawn: resolve the kill vs save, log the investigation, announce, check win. */
function resolveNight(state: MafiaState, now: number): MafiaState {
  const victim = plurality(state.night.mafiaVotes, state.seed)
  const saved = victim !== null && victim === state.night.doctorSave
  const killedId = victim !== null && !saved ? victim : null

  const alive = { ...state.alive }
  if (killedId) alive[killedId] = false

  const detectiveLog = [...state.detectiveLog]
  if (state.night.detectiveCheck) {
    detectiveLog.push({
      targetNickname: nickname(state, state.night.detectiveCheck) ?? '?',
      isMafia: state.roles[state.night.detectiveCheck] === 'mafia',
    })
  }

  const next: MafiaState = {
    ...state,
    alive,
    detectiveLog,
    lastNight: { killedId, saved: victim !== null && saved },
    night: { mafiaVotes: {}, doctorSave: null, detectiveCheck: null },
    votes: {},
    phase: 'day',
    deadline: now + state.settings.discussionSeconds * 1000,
  }
  const winner = checkWinner(next)
  return winner ? { ...next, phase: 'finished', winner, deadline: null } : next
}

/** Dusk vote resolution: strict plurality eliminates; any tie spares everyone. */
function resolveVote(state: MafiaState, now: number): MafiaState {
  const counts = new Map<string, number>()
  for (const t of Object.values(state.votes)) counts.set(t, (counts.get(t) ?? 0) + 1)
  const max = Math.max(0, ...counts.values())
  const top = [...counts.entries()].filter(([, n]) => n === max).map(([id]) => id)
  const eliminatedId = max > 0 && top.length === 1 ? top[0] : null

  const alive = { ...state.alive }
  if (eliminatedId) alive[eliminatedId] = false

  return {
    ...state,
    alive,
    lastVote: {
      eliminatedId,
      revealedRole: eliminatedId ? state.roles[eliminatedId] : null,
      tally: [...counts.entries()]
        .map(([id, count]) => ({ nickname: nickname(state, id) ?? '?', count }))
        .sort((a, b) => b.count - a.count),
    },
    phase: 'reveal',
    deadline: now + state.settings.revealSeconds * 1000,
  }
}

/** Leave reveal: win check, else a new night falls. */
function toNight(state: MafiaState, now: number): MafiaState {
  const winner = checkWinner(state)
  if (winner) return { ...state, phase: 'finished', winner, deadline: null }
  return {
    ...state,
    phase: 'night',
    day: state.day + 1,
    votes: {},
    deadline: now + state.settings.nightSeconds * 1000,
  }
}

function baseView(state: MafiaState): BaseView {
  return {
    day: state.day,
    players: state.players.map((p) => ({ id: p.id, nickname: p.nickname, alive: state.alive[p.id] })),
  }
}
function allRoles(state: MafiaState) {
  return state.players.map((p) => ({ nickname: p.nickname, role: state.roles[p.id] }))
}
function lastNightView(state: MafiaState) {
  return {
    killedNickname: nickname(state, state.lastNight?.killedId ?? null),
    saved: state.lastNight?.saved ?? false,
  }
}

/** Classic Mafia. All hidden information is enforced here, never in clients. */
export const mafia: GameDefinition<MafiaState, MafiaSettings, MafiaPrompt> = {
  id: 'mafia',
  minPlayers: 4,
  defaultSettings: { nightSeconds: 45, discussionSeconds: 150, voteSeconds: 45, revealSeconds: 12 },

  init({ players, settings, now }) {
    return {
      phase: 'night',
      day: 1,
      players,
      roles: assignRoles(players, now),
      alive: Object.fromEntries(players.map((p) => [p.id, true])),
      night: { mafiaVotes: {}, doctorSave: null, detectiveCheck: null },
      lastNight: null,
      detectiveLog: [],
      votes: {},
      lastVote: null,
      winner: null,
      seed: now,
      settings,
      deadline: now + settings.nightSeconds * 1000,
    }
  },

  reducer(state, action): MafiaState {
    switch (action.type) {
      case 'PLAYER_INPUT': {
        const pid = action.playerId
        if (!state.players.some((p) => p.id === pid) || !state.alive[pid]) return state
        const target = (action.input as { targetId?: unknown })?.targetId
        if (typeof target !== 'string' || !state.alive[target]) return state

        if (state.phase === 'night') {
          const role = state.roles[pid]
          const night = { ...state.night }
          if (role === 'mafia') {
            if (state.roles[target] === 'mafia') return state // no friendly fire
            night.mafiaVotes = { ...night.mafiaVotes, [pid]: target }
          } else if (role === 'doctor') {
            night.doctorSave = target // self-save allowed (classic)
          } else if (role === 'detective') {
            if (target === pid) return state
            night.detectiveCheck = target
          } else {
            return state // civilians sleep
          }
          const next = { ...state, night }
          return nightComplete(next) ? resolveNight(next, action.now) : next
        }

        if (state.phase === 'vote') {
          if (target === pid) return state
          const votes = { ...state.votes, [pid]: target }
          const next = { ...state, votes }
          const voters = aliveIds(next).filter((id) => next.players.find((p) => p.id === id)?.connected)
          return voters.every((id) => votes[id]) ? resolveVote(next, action.now) : next
        }
        return state
      }
      case 'TIMER_EXPIRED':
      case 'HOST_ADVANCE':
        if (state.phase === 'night') return resolveNight(state, action.now)
        if (state.phase === 'day') return { ...state, phase: 'vote', deadline: action.now + state.settings.voteSeconds * 1000 }
        if (state.phase === 'vote') return resolveVote(state, action.now)
        if (state.phase === 'reveal') return toNight(state, action.now)
        return state
    }
  },

  playerView(state, playerId): MafiaPlayerView {
    const role = state.roles[playerId]
    const base = {
      ...baseView(state),
      role,
      isAlive: state.alive[playerId] === true,
      // Mafia know their team; the detective carries their case file. Nobody else gets extras.
      ...(role === 'mafia'
        ? { mafiaTeam: state.players.filter((p) => state.roles[p.id] === 'mafia').map((p) => p.nickname) }
        : {}),
      ...(role === 'detective' ? { detectiveLog: state.detectiveLog } : {}),
    }

    if (state.phase === 'night') {
      const action =
        !state.alive[playerId] ? null
        : role === 'mafia' ? ('kill' as const)
        : role === 'doctor' ? ('save' as const)
        : role === 'detective' ? ('investigate' as const)
        : null
      const candidates =
        action === null
          ? []
          : state.players
              .filter((p) => state.alive[p.id])
              .filter((p) => (role === 'mafia' ? state.roles[p.id] !== 'mafia' : true))
              .filter((p) => (role === 'detective' ? p.id !== playerId : true))
              .map(({ id, nickname: n }) => ({ id, nickname: n }))
      const yourTarget =
        role === 'mafia' ? state.night.mafiaVotes[playerId] ?? null
        : role === 'doctor' ? state.night.doctorSave
        : role === 'detective' ? state.night.detectiveCheck
        : null
      return { ...base, phase: 'night', action, candidates, yourTarget, deadline: state.deadline }
    }
    if (state.phase === 'day') {
      return { ...base, phase: 'day', lastNight: lastNightView(state), deadline: state.deadline }
    }
    if (state.phase === 'vote') {
      return {
        ...base,
        phase: 'vote',
        candidates: state.players
          .filter((p) => state.alive[p.id] && p.id !== playerId)
          .map(({ id, nickname: n }) => ({ id, nickname: n })),
        yourVote: state.votes[playerId] ?? null,
        deadline: state.deadline,
      }
    }
    if (state.phase === 'reveal') {
      return {
        ...base,
        phase: 'reveal',
        eliminatedNickname: nickname(state, state.lastVote?.eliminatedId ?? null),
        revealedRole: state.lastVote?.revealedRole ?? null,
        tally: state.lastVote?.tally ?? [],
      }
    }
    return { ...base, phase: 'finished', winner: state.winner ?? 'town', allRoles: allRoles(state) }
  },

  hostView(state): MafiaHostView {
    const base = baseView(state)
    if (state.phase === 'night') {
      const needed =
        aliveWithRole(state, 'mafia').length +
        aliveWithRole(state, 'doctor').length +
        aliveWithRole(state, 'detective').length
      const done =
        Object.keys(state.night.mafiaVotes).length +
        (state.night.doctorSave ? 1 : 0) +
        (state.night.detectiveCheck ? 1 : 0)
      return { ...base, phase: 'night', actionsDone: done, actionsNeeded: needed, deadline: state.deadline }
    }
    if (state.phase === 'day') return { ...base, phase: 'day', lastNight: lastNightView(state), deadline: state.deadline }
    if (state.phase === 'vote') {
      const voters = aliveIds(state).length
      return { ...base, phase: 'vote', votedCount: Object.keys(state.votes).length, totalVoters: voters, deadline: state.deadline }
    }
    if (state.phase === 'reveal') {
      return {
        ...base,
        phase: 'reveal',
        eliminatedNickname: nickname(state, state.lastVote?.eliminatedId ?? null),
        revealedRole: state.lastVote?.revealedRole ?? null,
        tally: state.lastVote?.tally ?? [],
      }
    }
    return { ...base, phase: 'finished', winner: state.winner ?? 'town', allRoles: allRoles(state) }
  },

  isFinished: (state) => state.phase === 'finished',
}
```

## Deviations

- Task 1 targets the current engine contract, which now includes `PLAYER_CONNECTION_CHANGED`. The Mafia reducer therefore updates its copied player connection data immutably and re-evaluates night/vote completion after each real connection change.
- Completion, host `actionsNeeded`/`actionsDone`, and host `totalVoters`/`votedCount` use connected living participants. Stored actions and votes from disconnected participants are excluded from both progress and resolution while they remain disconnected. A phase never advances merely because its connected actor/voter set is empty.
- Connected living players may revise a valid night target or day vote while the phase remains open; the latest accepted input wins. The final required input resolves the phase immediately, so later inputs are naturally ignored by the next phase.
- Player views use the reducer's same living-and-connected eligibility rule: spectators retain permitted role/history information but receive no night action, night target, vote target, or candidate controls. Unknown player IDs are rejected explicitly rather than producing a view with an undefined role.
- Mafia night plurality uses target-ID lexical order after an equal vote count and equal seeded hash, so even adversarial hash collisions resolve independently of vote iteration order.
